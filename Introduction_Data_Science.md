# Introduction Data Science

## Welcome

Welcome to the Canvas page for Introduction to Data Science. In this course, we will dive into the world of data science. The course tries to give a broad overview of important concepts that will be useful in other, more advanced, data science courses. However, we will also provide various examples and illustrations of data science applications. You will learn how to use programming software (we focus on R) to implement these illustrations.
